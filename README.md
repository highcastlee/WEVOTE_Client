## ✨ Features
- 선거 단위별 검색 및 상세 페이지 이동
  - 상세 페이지 : 후보자 정보 및 공약 정보, Q&A 기능 제공
- 관리자 페이지를 통해 등록된 선거 단위별 요약 정보 제공
- 메인 배너 이미지
- 선거 공지사항 및 안내 페이지
- 로컬 회원가입/로그인
- 관리자 페이지 모든 선거 정보 관련 CRUD 기능
